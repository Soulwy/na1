## 📖 О проекте

**Book Tracker** — это легковесное веб-приложение для организации личной библиотеки и контроля процесса чтения. В отличие от популярных сервисов (Goodreads, StoryGraph, Bookly), приложение не требует регистрации, работает полностью офлайн и хранит все данные локально в браузере пользователя.

Проект разработан в рамках курсовой работы и демонстрирует применение современных веб-технологий для создания автономных клиентских приложений.

---

## ✨ Основные возможности

### 📚 Управление книгами (CRUD)
- Добавление новых книг с указанием названия, автора, количества страниц, даты начала чтения и дневной цели
- Редактирование информации через модальное окно
- Удаление книг с автоматическим обновлением интерфейса и хранилища
- Просмотр всех книг в виде информативных карточек

### 📊 Отслеживание прогресса
- Указание количества прочитанных страниц
- Автоматический расчёт процента выполнения чтения
- Визуальное отображение прогресса (текстовый индикатор и прогресс-бар)
- Автоматическая смена статуса на «Прочитано» при достижении 100%

### 🏷️ Система статусов
- **Читаю** — книга в процессе чтения
- **Прочитано** — книга полностью прочитана
- **Отложено** — чтение временно приостановлено
- Быстрое переключение статусов через кнопки на карточке

### ⭐ Избранное
- Добавление книг в избранное для быстрого доступа
- Визуальное обозначение избранных книг золотистой полосой слева и иконкой ⭐

### 🔍 Фильтрация и поиск
- Фильтрация книг по статусу (все / читаю / прочитано / отложено)
- Динамический поиск по названию и автору без перезагрузки страницы

### 📈 Статистика
Автоматический расчёт и отображение ключевых показателей:
- Общее количество книг
- Количество завершённых книг
- Средний процент прочтения
- Средняя скорость чтения (стр/день)

### 🌓 Темы оформления
- Поддержка светлой и тёмной темы
- Переключение в один клик (🌙/☀️)
- Сохранение выбранной темы между сеансами

### 💾 Локальное хранение данных
- Все данные сохраняются в LocalStorage браузера
- Автоматическое восстановление данных при повторном открытии
- Работа без подключения к интернету

## 🎁 Демо-данные

При первом запуске приложение автоматически создаёт 3 демонстрационные книги:

| Книга | Автор | Страниц | Статус |
|-------|-------|---------|--------|
| Мастер и Маргарита | Булгаков | 480 | Читаю (120 стр.) ⭐ |
| JavaScript для детей | Ник Морган | 320 | Прочитано |
| Чистый код | Роберт Мартин | 464 | Отложено |

Демо-данные создаются однократно и не перезаписываются при повторных запусках.
